DEMO演示 html5新增结构标签


第一部分：新结构标签 用上这些标签语义化更好 SEO更友好。如果低版本浏览器想兼容结构化标签 可以使用html5shiv.js
header
nav
section定义一个区域
article定义一篇文章
文章标题header
文章页脚footer
aside
footer



第二部分：表单元素不再需要嵌套在form里面 元素只需要加上from="表单id"就能对应到某个表单
新input类型 新表单元素 新表单属性 支持度太差 基本不用


第三部分：常用被删除的标签
frameset frame noframes被删除 都用iframe
框架集SEO很差 搜索引擎不会抓取框架集内容 所以框架集一般用于不需抓取的后台设计

		figure定义图文区域
			<figcaption>figcaption图文标题</figcaption>

			<meter value="4" min="0" max="10" low="3" high="7"></meter> 4/10 meter IE不支持 3以下7以上变色<br/>
			<progress value="25" max="100" id="pro"></progress> 25% progress IE10+支持<br/>
			当前进度<span id="pro_num">25</span>% <input type="button" id="btn1" value="开始进度条">
			<script type="text/javascript">
				var oBtn1 = document.getElementById('btn1');
				var oPro = document.getElementById('pro');
				var oProNum = document.getElementById('pro_num');
				var Timer = null;
				oBtn1.onclick = function(){
					clearInterval(Timer);
					Timer = setInterval(function(){
						oPro.value = oPro.value + 1;
						oProNum.innerHTML = oPro.value;
						if(oPro.value == '100'){
							clearInterval(Timer);
						}
					}, 50);
				}
			</script>
		</figure>
